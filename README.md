Veitool V1.0.3 正式版
===============
Veitool 后台框架系统，基于ThinkPHP8.x + Layui2.8.x、Laravel10.x + Layui2.8.x架构，拥有完善的菜单、角色、权限管理机制，集成了常用的管理构建模块，用户可快速便捷的在此基础上进行二次开发、配置、扩展功能，以及可通过安装插件快捷实现系统功能。
